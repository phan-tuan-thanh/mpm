# Hướng dẫn sử dụng Agile PM

Chào mừng bạn đến với tài liệu hướng dẫn sử dụng **Agile PM** — nền tảng quản lý dự án Agile toàn diện dành cho các đội phát triển phần mềm.

## Mục lục

| # | Tài liệu | Mô tả |
|---|----------|--------|
| 1 | [Bắt đầu nhanh](./01-bat-dau-nhanh.md) | Đăng nhập, tổng quan giao diện, tạo dự án đầu tiên |
| 2 | [Quản lý dự án](./02-quan-ly-du-an.md) | Tạo, tìm kiếm, lọc và xóa dự án |
| 3 | [Work Items (Tasks)](./03-work-items.md) | Tạo task, Kanban board, danh sách, kéo thả, bulk actions |
| 4 | [Chi tiết Task](./04-chi-tiet-task.md) | Mô tả, sub-items, attachments, links, activity log |
| 5 | [Sprints](./05-sprints.md) | Tạo, bắt đầu, hoàn thành sprint; dashboard và velocity |
| 6 | [Modules](./06-modules.md) | Nhóm tasks thành phân hệ logic |
| 7 | [Cài đặt dự án](./07-cai-dat-du-an.md) | Cấu hình chung, thành viên, tính năng, danger zone |
| 8 | [Quản trị hệ thống](./08-quan-tri-he-thong.md) | Quản lý user, phân quyền (dành cho Admin) |

## Yêu cầu hệ thống

- Trình duyệt: Chrome, Firefox, Edge hoặc Safari phiên bản mới nhất
- Độ phân giải tối thiểu: 1280×720 (desktop), 768px (tablet)
- Kết nối internet ổn định

## Quy ước trong tài liệu

| Ký hiệu | Ý nghĩa |
|----------|---------|
| 🔒 | Yêu cầu quyền đặc biệt (Admin hoặc Scrum Master) |
| ⚠️ | Hành động không thể hoàn tác |
| 💡 | Mẹo sử dụng |
| 📌 | Lưu ý quan trọng |
